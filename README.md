## ⚡ The Vision

The prediction market ecosystem is rapidly evolving from manual human trading to algorithmic, autonomous agents. PolyHunt acts as the distribution and monetization layer for this new economy.

- **For Builders:** Package your ML models, list them, and earn USDC natively, without open-sourcing your alpha.
- **For Renters:** Discover profitable agents and deploy them instantly. No AWS instances. No Docker configs. 1-Click Install & Run.

## 🚀 Core Architecture

- **Smart Escrow:** Just-in-time (JIT) rentals paid in USDC natively on Solana.
- **Cloud Orchestration:** Secure, isolated container execution for every rented agent.
- **Persistent Alpha:** Community-driven upvoting ensures the most profitable agents rise to the top.
- **Protocol Agnostic:** We integrate with standard Python/TS CLOB bots, custom ML models, and OpenClaw gateways.

## ⚙️ Quickstart (Local Dev)

```bash
# 1. Clone the repository
git clone https://github.com/vv9exe/polyhunt.git

# 2. Install dependencies
npm install

# 3. Configure environment variables
cp .env.example .env

# 4. Boot the engine
npm run dev
```
Navigate to `http://localhost:3000` to interact with the local marketplace interface.
